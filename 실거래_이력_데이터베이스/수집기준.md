# 실거래 데이터 수집 기준

> 최종 갱신: 2026-03-09

## 1. 수집 원칙

### 면적 기준: 반드시 확인 후 기록
- **공급면적** 기준이면 공급면적으로 통일
- **전용면적** 기준이면 전용면적으로 통일
- **혼재 금지**: 한 CSV 안에서 공급/전용이 섞이면 안 됨
- 현재 CSV 기준: **공급면적** (네이버 부동산 면적 탭 기준, 2026-03-08 스크린샷 검증)

### 면적 출처별 기준
| 출처 | 표시 기준 | 비고 |
|:---|:---|:---|
| **네이버 부동산** (land.naver.com) | 공급면적 | 면적 탭/필터가 공급 기준 |
| **아실** (asil.kr) | 확인 필요 | 매번 확인 후 사용 |
| **국토부 실거래** (rt.molit.go.kr) | 전용면적 | 다운로드 시 전용/공급 컬럼 별도 |

### ⚠️ 수집 전 체크리스트
1. [ ] 사이트에서 면적이 전용인지 공급인지 **화면에서 확인**
2. [ ] CSV의 기존 area_m2 기준과 **동일한 기준** 확인
3. [ ] 가격이 해당 평형의 시세 범위 내인지 **상식 검증**
4. [ ] 상충/상이 발견 시 **사용자 확인** 후 기록

## 2. 기간 기준

- **기본값**: 특정 기간 미지정 시 → **최근 실거래가 적용** (가장 최근 거래 기준, 기간 제한 없음)
- 유사거래 비교법: 가장 최근 + 가장 유사한 층수의 거래가 우선 적용
- 사용자가 기간을 명시하면 해당 기간 내 거래만 적용
- 예: "최근 3개월" → 3개월 내 거래만 표시
