## PART 2. 학술 논문 기반 핵심 인사이트 (출처 명시)

### 2-1. 손실회피 (Loss Aversion)
> **Genesove & Mayer (2001)**, Quarterly Journal of Economics [SSCI]
- 손실 구간 매도자 → 호가 **25~35% 과대 설정**
- 결과: 판매 기간 **2배 연장**, 최종 매매가 **오히려 하락**
- **멘트**: "연구에 따르면 시장가보다 높은 호가를 유지하면 판매 기간은 2배,최종 매매가는 오히려 낮아집니다"

### 2-2. 앵커링 (Anchoring)
> **Northcraft & Neale (1987)**, OBHDP [SSCI]
- 부동산 전문가조차 최초 제시가격의 영향 **41% 이상** 받음
- **멘트**: "먼저 데이터 기반 가격을 제시하면 협상 범위를 통제할 수 있습니다"

### 2-3. 프레이밍 (Framing)
> **Kahneman & Tversky (1979)**, Prospect Theory [SSCI]
- 사람은 이익보다 **손실에 2배 강하게** 반응
- **멘트**: 항상 "안 하면 잃는 것"을 먼저 → "하면 얻는 것"은 부연

### 2-4. 부여효과 (Endowment Effect)
> **박순미·이성화 (2025)**, 한국지적학회지 [KCI]
- 소유자는 시장가 대비 **2~3배 높게** 자산 평가
- **멘트**: "시장이 보는 가격과 주인의 애정은 별개입니다. 데이터로 보여드리겠습니다"

## PART 3. 실시간 수집 시스템 운용 가이드

### 3-1. 공공데이터포털 수집기 (public_data_collector.py)

| API | 데이터 | 명령어 | 활용 |
|:---|:---|:---|:---|
| 국토부 실거래가 | 유니시티/용지아이파크/힐스테이트 3단지 비교 | `--trade` | BATNA 구축, 앵커링 데이터 |
| 한국은행 ECOS | 기준금리, 대출금리, 소비자물가 | `--rate` | 보유비용 계산, 매수 타이밍 |
| KOSIS | 창원시 인구이동, 세대수 변화 | `--population` | 수요/공급 예측 |

```
# API 키 설정 (최초 1회)
python 협상_인텔리전스\실시간수집\public_data_collector.py --setup

# 실거래가 수집 (월 1회)
python 협상_인텔리전스\실시간수집\public_data_collector.py --trade

# 금리 수집 (금통위 회의 후)
python 협상_인텔리전스\실시간수집\public_data_collector.py --rate

# 전체 수집
python 협상_인텔리전스\실시간수집\public_data_collector.py --all
```

**API 키 발급 사이트**:
- data.go.kr → "아파트매매 실거래 상세 자료" 활용신청
- ecos.bok.or.kr → Open API → 인증키 신청 (즉시 발급)
- kosis.kr → Open API → 인증키 신청

### 3-2. 네이버 데이터랩 수집기 (naver_datalab_collector.py)

| 키워드 그룹 | 비교 | 활용 |
|:---|:---|:---|
| 유니시티 매매 vs 전세 | 매매/전세 수요 비교 | 시장 방향성 판단 |
| 창원 아파트 매매 vs 전세 | 지역 전체 수급 | 거시 수요 파악 |
| 유니시티 vs 용지아이파크 vs 힐스테이트 | 단지 간 관심도 | 경쟁 매물 분석 |

```
# API 키 설정 (최초 1회)
python 협상_인텔리전스\실시간수집\naver_datalab_collector.py --client-id YOUR_ID --client-secret YOUR_SECRET --save-config

# 트렌드 수집 (주 1회 또는 특정 시기 즉시)
python 협상_인텔리전스\실시간수집\naver_datalab_collector.py
```

**자동 생성 멘트 예시**:
- 매매 검색량 급증 시: "유니시티 매매 검색량이 25% 급증. 매수 대기 수요가 움직이는 선행 지표입니다"
- 전세 검색량 급증 시: "전세 수요 확대. 매매 전환 수요 유입 가능성 높습니다"
