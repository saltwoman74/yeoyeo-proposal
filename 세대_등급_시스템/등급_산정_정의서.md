# 창원 유니시티 6,100세대 정밀 등급 산정 체계 (Grading System)

이 문서는 창원 유니시티 1~4단지 총 6,100세대의 가치를 개별적으로 평가하기 위한 **공학적 등급 산정 모델**을 정의합니다. 본 모델은 부동산 감정평가 이론과 실무 협상 데이터를 결합하여 설계되었습니다.

---

## 1. 등급 분류 및 평가 항목 (Grading Categories)

### [A] 입지 등급 (Location Grade - L) : 고정적 요소
*변경 불가능한 물리적 위치 가치를 평가합니다.*
1.  **조망권 (View):** 중앙공원 영구 조망, 사화공원 조망, 단지 내 조경 조망, 시티뷰.
2.  **일조/향 (Orientation):** 정남향, 남동향, 남서향 등 (일조 시간 분석).
3.  **층수 (Floor):** 로열층(중상층), 저층(테라스/필로티), 최상층(펜트하우스 등).
4.  **접근성 (Accessibility):** 스트리트몰(어반브릭스) 인접성, 초등학교(의창초) 통학 거리, 주출입구 인접도.
5.  **소음/프라이버시:** 대로변 소음 노출 정도 및 동간 거리 확보 수준.

### [B] 상태 등급 (Condition Grade - C) : 가변적 요소
*세대 내부의 물리적 상태와 옵션 수준을 평가합니다.*
1.  **확장 및 구조:** 발코니 확장 여부, 팬트리/드레스룸 특화 옵션.
2.  **내부 옵션:** 시스템 에어컨(개수), 주방 상판(엔지니어드 스톤 등), 바닥재(폴리싱 타일/마루).
3.  **유지 관리:** 실거주 여부(관리 양호), 임차인 거주(노후도), 최근 인테리어 리모델링 여부.
4.  **하자 상태:** 준공 후 연차에 따른 누수, 결로, 설비 상태 (공식 AS 이력 기반).

### [C] 상황 등급 (Situation Grade - S) : 유동적 요소
*협상력에 직접적인 영향을 미치는 심리적·경제적 배경을 평가합니다.*
1.  **매도 시급성:** 일시적 2주택 비과세 기한, 분양권 전매 제한, 개인적 급전 필요성.
2.  **점유 형태:** 공실(즉시 입주 가능 - 최고점), 주인 거주, 임차인 만기 임박, 장기 임대차 유지.
3.  **가격 유연성:** 협상 가능 범위(Nego Range), 매도인의 심리적 저항선(Target Price).
4.  **세무 상황:** 양도세 비과세 요건 충족 여부, 법인 매물 여부.

---

## 2. 종합 등급 산정 공식 (Overall Score)

종합 점수($GS$)는 각 항목의 가중치를 적용하여 산출합니다.

$$GS = (L \times 0.5) + (C \times 0.3) + (S \times 0.2)$$

*   **L (입지):** 가격 하방 경직성을 결정하는 핵심 (50%)
*   **C (상태):** 실거주 가점 및 즉시 비용 절감 요소 (30%)
*   **S (상황):** 실제 계약 체결 가능성과 가격 결정의 변수 (20%)

---

## 3. 데이터 구조 설계 (Data Schema)

6,100세대의 데이터를 관리할 JSON/CSV 표준 규격입니다.

| 항목(Key) | 설명 | 데이터 타입 | 예시 |
| :--- | :--- | :--- | :--- |
| `Unit_ID` | 단지-동-호수 | String | `U1-101-1502` |
| `L_Score` | 입지 점수 (1-10) | Float | `9.5` |
| `C_Score` | 상태 점수 (1-10) | Float | `8.0` |
| `S_Score` | 상황 점수 (1-10) | Float | `7.0` |
| `Total_Grade` | 최종 등급 | String | `S+ (Premium)` |
| `Key_Features` | 주요 특징 요약 | Array | `["공원조망", "풀옵션", "급매"]` |
| `Analysis_Date` | 최종 분석 일자 | Date | `2026-03-04` |

---

## 4. 향후 분석 고도화 계획 (Continuous Analysis)

1.  **실거래가 연동:** 실시간 국토부 실거래가 데이터를 바탕으로 등급별 적정 가격 도출.
2.  **AI 예측 모델:** 등급 변화 추이에 따른 향후 가치 상승 가능성(Probable Appreciation) 산출.
3.  **협상 시뮬레이션:** 상황 등급(S)에 따른 맞춤형 협상 Script 생성 연동.

---
**주의:** 본 시스템의 모든 데이터 입력은 정부 공식 데이터(공시가격, 실거래가)와 현장 검증 자료만을 근거로 하며, 주관적 편향을 최소화하기 위해 '부동산 감정평가 기준'을 준거합니다.
